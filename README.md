# mGPUKDE
Multi-GPU Parallel Kernel Density Estimation for Spatial Point Pattern Analysis.
## Citation
Zhang G. (2022). Detecting and visualizing observation hot-spots in massive volunteer-contributed geographic data across spatial scales using GPU-accelerated kernel density estimation. ISPRS International Journal of Geo-Information, 11(1): 55. https://doi.org/10.3390/ijgi11010055
## Dependencies
1. CUDA 11.4: https://developer.nvidia.com/cuda-11-4-0-download-archive.
2. GDAL: https://gdal.org/index.html.
## IDE
Visual Studio 2019.
